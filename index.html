    Imprint CMS

    Imprint CMS is a domain specific CMS for publishers, based on ASP.net MVC 3 and LinqToSQL. It does very little, but tries to do it well.

    Tools

    Imprint was developed using the free Express tools from Microsoft (Visual Web Developer 2010 Express and SQL Server 2008 R2 Express).

    Get running

    There is an empty database backup in the repository. Restore to your server of choice. The code refers by default to the SQL Express instance on localhost using Windows authentication. Then fire up the project and run it. To install properly, use Web Deploy to transform the web.config, then replace the dummy values for connection string etc. to match your environment.
